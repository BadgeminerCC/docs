Functions

init (silent)
Initializes the RedRun runtime. This is called automatically, but it's still available if desired.

Parameters:

  • silent Set to any truthy value to inhibit the status message.
start (func, name)
Starts a coroutine running in the background.

Parameters:

  • func The function to run.
  • name A value to use to identify this task later. Can be any value, including nil/none.

Returns:

    The ID of the started task.
getid (name)
Returns the task ID for a named task.

Parameters:

  • name The name of the task.

Returns:

    The ID of the task with the name, or nil if not found.
getstate (id)
Returns a table of custom state held by the task.

Parameters:

  • id The ID of the task.

Returns:

    The state table held by the task, or nil if the task doesn't exist.
kill (id)
Kills a task immediately.

Parameters:

  • id The ID of the task.
terminate (id)
Terminates a task. This sends it one terminate event, and then removes it from the queue.

Parameters:

  • id The ID of the task.
